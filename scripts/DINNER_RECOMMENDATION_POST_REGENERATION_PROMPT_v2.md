## 0. 망포역 포스트 평가 (v1 대비 v2가 개선한 부분)

### 망포역 가성비 포스트의 강점 (보존할 부분)

기존 망포역 포스트가 강남역 치킨 포스트보다 개선된 부분:
- ✓ 데이터 출처 섹션 추가 (행정안전부·관광공사·식약처 명시)
- ✓ 운영자 노트 섹션 추가 ("345곳 데이터에서 필터링")
- ✓ 선정 기준 섹션 (투명함 향상)
- ✓ 상황별 추천 + 체크포인트 + FAQ 구조
- ✓ 태그 구조 (#망포 #가성비 #점심)
- ✓ 도입부 통계 (평균 4.1점, 1,687건)

이 강점들은 모범 답안에서도 그대로 유지합니다.

### 망포역 가성비 포스트의 치명적 문제 6가지 (해결할 부분)

**1. 반복 문구 (강남역 치킨과 동일 패턴)**

5개 식당 설명에서 동일 문구 반복:
- "같은 카테고리의 다른 후보들과 비교했을 때 신뢰도 측면에서 우위에 있습니다." → 5번 반복
- "전용 주차장이 없는 편이라 대중교통 이용을 권장합니다." → 5번 반복
- "평가가 카테고리 평균 수준이라, 단골 식당으로 가볍게 들르기 좋은 위치입니다." → 3번 반복

**해결**: 각 식당별 차별화 묘사 (시그니처 메뉴·운영 특성·가격 위치 중심)

**2. 데이터 모순 (가장 큰 신뢰도 손실)**

- 행복한김밥: 평점 3.4점 (호불호 갈리는)
- 그런데 같은 문장: "신뢰도 측면에서 우위에 있습니다"
- 평점 3.4가 어떻게 신뢰도 우위? 시스템 로직 오류

**해결**: 평점·리뷰 수치에 따라 분기 처리. 낮은 평점 식당은 "신뢰도 우위" 표현 금지.

**3. 영업시간 데이터 오류**

- 깡우동: "11:00 AM~12:00 PM" (1시간 영업?)
- 서천교동짬뽕: "휴무일" (영업 안 함?)

**해결**: 데이터 검증 시스템. 영업 1시간 표시·"휴무일" 단독 표시는 자동 거절.

**4. 비교표 한줄평 빈약**

- "칼국수 전문", "가격 대비 양 많음" 등 5개가 거의 같은 패턴
- 깡우동인데 "칼국수 전문" (실제는 우동 전문점)

**해결**: 한줄평은 식당의 진짜 차별화 1가지로 (가격대 위치 · 검증 신뢰 · 영업 특성 · 메뉴 다양성)

**5. 카테고리 정의 불명확**

- "가성비" = 1만원대 점심? 일부 식당 12,000~20,000원
- "1만원대"는 10,000~19,999원? 10,000~10,999원?

**해결**: 가격 범위 명시. 8,000원~13,000원 = 1만원대 점심 표준 범위 정의.

**6. 행복한김밥 추천 모순**

- 평점 3.4 (낮음)
- 그런데 "한 끼 만원대 가성비 점심"에 추천
- 다른 식당과 똑같이 "신뢰도 우위"

**해결**: 평점 낮은 식당은 차별화 포인트 (예: 06:00 영업·분식 카테고리 대표) 명시. 솔직히 한계 짚기.

## 4. 식당별 차별화 묘사 자동 생성 규칙

### 4-A. 식당별 차별화 포인트 자동 식별

각 식당의 진짜 강점 1~2가지 자율 식별:

**강점 카테고리 (자동 분류)**:
- 가격 위치 (저렴·평균·고가 vs 카테고리 평균)
- 검증 신뢰 (리뷰 수 표본 크기)
- 평점 우위 (카테고리 평균 대비)
- 운영 안정성 (운영 기간·인허가 검증)
- 영업 차별화 (이른 시간·늦은 시간·단체석·예약 등)
- 메뉴 차별화 (시그니처 메뉴·단일 메뉴 회전·세트 메뉴)

각 식당이 가진 강점 1~2가지 자동 식별 후, 차별화 묘사 자동 생성.

### 4-B. 차별화 묘사 자동 생성 패턴 예시

**예시 1: 가격 위치 중심**
> "{지역} 가성비 평균(10,400원)보다 **{차이}원 저렴/비싼** 식당입니다. {시그니처 메뉴}가 {가격}이라 가성비 1순위로 묶기 좋습니다."

**예시 2: 검증 신뢰 중심**
> "리뷰 {N}건은 {지역} {카테고리} 카테고리 {순위} 표본입니다. 평균값을 신뢰할 수 있는 안정 식당입니다."

**예시 3: 평점 우위 중심**
> "리뷰 {N}건은 표본이 작아 평균값 신뢰도가 다른 식당보다 낮습니다. 다만 **평점 {N}점은 {곳} 중 최고**이며, 새로 발견하기 좋은 후보로 묶을 수 있습니다."

**예시 4: 솔직한 한계 + 차별점**
> "평점 {N}점은 {곳} 중 가장 낮습니다. 호불호가 갈리는 식당이라 메뉴 선택이 중요합니다. 다만 **{시간} 영업**으로 {지역}에서 가장 이른 시간에 여는 {카테고리}점입니다."

같은 식당 묘사 패턴이 다른 식당에 반복 X. 식당마다 자동 분기.

### 4-C. 솔직한 한계 명시 강제

평점 4.0 미만 식당이 포함되면 한계 솔직히 명시:
- "평점 N점은 5곳 중 가장 낮습니다"
- "호불호가 갈리는 식당이라 메뉴 선택이 중요합니다"

다만 차별점 함께 명시:
- "다만 06:00 영업으로 망포에서 가장 이른 분식점입니다"
- "분식 카테고리 대표성을 위해 포함했습니다"
