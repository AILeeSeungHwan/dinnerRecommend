## 1. 등급 분류 기준

### A등급 — 유지
- 한 줄 결론이 intro에 명시됨
- 추천 식당 전부 data 파일에 실존
- 비교표/상황별 추천표 포함
- 데이터 기준일 명시
- 평점·리뷰수·가격 데이터 근거 있음
- 내부링크(식당 상세, 카테고리, AI추천)가 실제 존재하는 URL
- ending에 관련 글 3~5개 연결
- 리뷰 원문 미노출

### B등급 — 소폭 보강
- 구조는 괜찮음
- 데이터 기준일 갱신 필요 (오래된 날짜)
- 비교표는 있으나 상황별 추천표 누락
- callout/팁 부족
- 일부 식당 가격 정보 미갱신

### C등급 — 중간 리빌드
- 구조 불안정 (H2 부자연스러움)
- 추천 식당 중 폐업/이전 확인 필요
- 비교표 없이 나열만 한 구조
- 내부링크 일부 깨짐
- 데이터 기준일 3개월 이상 오래됨

### D등급 — 대폭 재작성
- 결론 없는 장문
- 리뷰 원문이 노출되어 있음
- 데이터 없이 주관적 추천만 존재
- AI 반복 문장 과다
- 추천 식당이 data 파일에 미존재

## 3. 포스트 유형별 추가 기준

### 카테고리 추천 포스트
- 5~7곳 추천 + 각 식당 데이터(평점, 가격, 메뉴) = A 필수
- 비교표 + 상황별 요약표 = A 필수

### 상황별 가이드 포스트
- 해당 상황 체크리스트(인원, 예산, 주차, 룸) = A 필수
- 3~5곳 추천 + 상황 적합 이유 = A 필수

### 비교형 포스트
- 동일 카테고리 3~5곳 직접 비교 테이블 = A 필수
- 1곳만 압도적 추천 금지 = A 필수
